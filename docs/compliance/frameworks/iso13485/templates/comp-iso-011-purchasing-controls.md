# COMP-ISO-011: Purchasing Controls

## Overview

**Purpose**: Purchasing control system implementation  
**ISO 13485 Clause**: 7.4 - Purchasing  
**Category**: purchasing, supplier-management  
**Priority**: High  
**Framework**: ISO 13485

## Description

Establishes purchasing controls to ensure purchased products conform to specified requirements and suppliers are evaluated and monitored.

## Definition of Done

- [ ] Purchasing procedures documented and approved
- [ ] Supplier evaluation and selection criteria established
- [ ] Supplier monitoring and re-evaluation processes implemented
- [ ] Purchasing information and verification procedures deployed
- [ ] Incoming inspection and acceptance processes established
- [ ] Staff training completed on purchasing control procedures
- [ ] Compliance validation performed and documented

---

_This requirement supports ISO 13485 Clause 7.4 compliance for purchasing._
