# COMP-ISO-014: Corrective and Preventive Action (CAPA)

## Overview

**Purpose**: Corrective and preventive action system implementation  
**ISO 13485 Clause**: 8.5.2, 8.5.3 - Corrective action, Preventive action  
**Category**: capa, improvement, nonconformity  
**Priority**: High  
**Framework**: ISO 13485

## Description

Establishes corrective and preventive action (CAPA) system to eliminate causes of nonconformities and prevent their recurrence or occurrence.

## Definition of Done

- [ ] CAPA procedures documented and approved
- [ ] Nonconformity investigation and root cause analysis processes implemented
- [ ] Corrective action planning and implementation procedures established
- [ ] Preventive action identification and implementation processes deployed
- [ ] CAPA effectiveness verification and monitoring systems implemented
- [ ] Staff training completed on CAPA procedures
- [ ] Compliance validation performed and documented

---

_This requirement supports ISO 13485 Clauses 8.5.2 and 8.5.3 compliance for corrective and preventive action._
