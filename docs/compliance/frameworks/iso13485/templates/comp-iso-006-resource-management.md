# COMP-ISO-006: Resource Management

## Overview

**Purpose**: Resource management system implementation  
**ISO 13485 Clause**: 6 - Resource management  
**Category**: resource-management, competence  
**Priority**: High  
**Framework**: ISO 13485

## Description

Establishes resource management processes to ensure adequate resources are provided for implementation and maintenance of the quality management system and regulatory compliance.

## Definition of Done

- [ ] Resource management procedures documented and approved
- [ ] Competency requirements defined for all roles
- [ ] Training and competency assessment programs implemented
- [ ] Infrastructure and work environment requirements established
- [ ] Resource allocation and monitoring processes deployed
- [ ] Staff training completed on resource management procedures
- [ ] Compliance validation performed and documented

---

_This requirement supports ISO 13485 Clause 6 compliance for resource management._
