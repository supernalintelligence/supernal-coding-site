# sc generate

Generate project files and documentation

## Usage

```bash
sc generate [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--templates` | Generate from templates |
| `--docs` | Generate documentation |
| `--workflows` | Generate workflow templates |

## Examples

```bash
# Basic usage
sc generate
```
