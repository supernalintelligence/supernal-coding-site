# sc agent

Agent workflow management

## Usage

```bash
sc agent [action] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `-c, --config <file>` | Configuration file path |
| `-v, --verbose` | Verbose output |
