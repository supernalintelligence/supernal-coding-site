# sc rules

Discover and manage repository rules

## Usage

```bash
sc rules [action] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `--scan` | Scan for rule files |
| `--validate` | Validate existing rules |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc rules
```
