# sc requirement

Requirement management system with API-like interface

**Alias:** `sc req`

## Usage

```bash
sc requirement [action] [...args] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |
| `[...args]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `-f, --format <type>` | Output format (json, table, csv) |
| `-v, --verbose` | Verbose output |
| `--no-color` | Disable colored output |

## Examples

```bash
# Basic usage
sc requirement

# Using alias
sc req
```
