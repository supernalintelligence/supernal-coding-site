# sc traceability

Traceability matrix for compliance and requirement tracking

**Alias:** `sc trace`

## Usage

```bash
sc traceability [action] [...args] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |
| `[...args]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `-f, --format <type>` | Output format (json, html, csv) |
| `-o, --output <path>` | Output file or directory |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc traceability

# Using alias
sc trace
```
