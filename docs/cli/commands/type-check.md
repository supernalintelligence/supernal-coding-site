# sc type-check

Detect and prevent TypeScript/JavaScript type duplications

## Usage

```bash
sc type-check [options]
```

## Options

| Option | Description |
|--------|-------------|
| `-f, --fix` | Automatically fix detected issues |
| `--report` | Generate detailed report |

## Examples

```bash
# Basic usage
sc type-check
```
