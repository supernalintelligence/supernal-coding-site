# sc git-assess

Comprehensive git repository assessment and analysis

**Alias:** `sc assess`

## Usage

```bash
sc git-assess [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--detailed` | Detailed analysis |
| `--format <type>` | Output format (json, table, markdown) |

## Examples

```bash
# Basic usage
sc git-assess

# Using alias
sc assess
```
