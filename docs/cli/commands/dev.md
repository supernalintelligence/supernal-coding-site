# sc dev

Development tools and utilities

## Usage

```bash
sc dev <action> [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `<action>` | (required) |

## Options

| Option | Description |
|--------|-------------|
| `-e, --environment <env>` | Environment to use |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc dev
```
