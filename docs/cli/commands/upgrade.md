# sc upgrade

Upgrade supernal-code to the latest version

## Usage

```bash
sc upgrade [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--force` | Force upgrade without confirmation |
| `--beta` | Install beta version |

## Examples

```bash
# Basic usage
sc upgrade
```
