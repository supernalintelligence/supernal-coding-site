# sc workflow

Project workflow management

## Usage

```bash
sc workflow [action] [...args] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |
| `[...args]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `-t, --template <name>` | Workflow template to use |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc workflow
```
