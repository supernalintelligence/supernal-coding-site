# sc config

Manage Supernal Coding configuration and privacy settings

## Usage

```bash
sc config [category] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[category]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `--global` | Global configuration |
| `--local` | Local configuration |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc config
```
