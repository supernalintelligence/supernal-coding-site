# sc test-map

Generate comprehensive test mapping and analysis

**Alias:** `sc tmap`

## Usage

```bash
sc test-map [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--output <file>` | Output file path |
| `--format <type>` | Output format (json, html, markdown) |

## Examples

```bash
# Basic usage
sc test-map

# Using alias
sc tmap
```
