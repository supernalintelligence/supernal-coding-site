# sc git-smart

Smart git workflow utilities

## Usage

```bash
sc git-smart [action] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `-v, --verbose` | Verbose output |
| `--dry-run` | Show what would be done without executing |

## Examples

```bash
# Basic usage
sc git-smart
```
