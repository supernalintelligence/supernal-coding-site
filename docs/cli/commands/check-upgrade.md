# sc check-upgrade

Check for available upgrades of supernal-code

## Usage

```bash
sc check-upgrade [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--force` | Force check even if recently checked |
| `--json` | Output in JSON format |

## Examples

```bash
# Basic usage
sc check-upgrade
```
