# sc sync

Synchronize local repository state with global sc installation
Actions: check, report, update

## Usage

```bash
sc sync [action] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `--force` | Force synchronization |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc sync
```
