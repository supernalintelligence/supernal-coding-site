# sc update

Update the global sc package installation

## Usage

```bash
sc update [action] [options]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `[action]` | (optional) |

## Options

| Option | Description |
|--------|-------------|
| `--force` | Force update |
| `--beta` | Update to beta version |

## Examples

```bash
# Basic usage
sc update
```
