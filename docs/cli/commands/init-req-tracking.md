# sc init-req-tracking

Initialize git tracking metadata for all requirement markdown files

**Alias:** `sc init-tracking`

## Usage

```bash
sc init-req-tracking [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--force` | Force initialization |
| `-v, --verbose` | Verbose output |

## Examples

```bash
# Basic usage
sc init-req-tracking

# Using alias
sc init-tracking
```
