# Tools

The Supernal Coding toolkit for compliant development.

## Core Tools

### [CLI (`sc`)](./cli.md)
Command-line interface for all operations.

```bash
sc requirement new "Feature"
sc validate
sc git-smart merge
```

### [Git Hooks](./git-hooks.md)
Automated quality gates on commit and push.

### [Dashboard](./dashboard.md)
Visual interface for requirement and compliance tracking.

## Quick Reference

| Command | Description |
|---------|-------------|
| `sc init` | Initialize repository |
| `sc requirement` | Manage requirements |
| `sc validate` | Run validation |
| `sc git-smart` | Safe git operations |
| `sc health` | Check repository health |
| `sc test` | Run tests |
