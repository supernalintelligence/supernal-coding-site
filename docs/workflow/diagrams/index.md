Visual representations of Supernal Coding workflows using Mermaid diagrams.

## Available Diagrams

- **[Workflow State Diagrams](./workflow-state-diagrams.md)** - Complete workflow state machines and decision trees

## Diagram Types

### State Diagrams
Show the lifecycle states of requirements, tasks, epics, and other workflow entities.

### Flow Diagrams
Visualize decision trees and process flows for common operations.

### Git Integration
Illustrate how git operations integrate with the workflow system.

### Compliance Flows
Show how compliance frameworks integrate with development workflows.
